# クリスタル・キャバーン・ダッシュ (Crystal Cavern Dash)

オリジナルの2Dサイドスクロール型プラットフォームゲームです。HTML / CSS / Vanilla JavaScript のみで実装されており、外部ライブラリ・画像・音声ファイルは使用していません。

## 実行方法

`index.html` をモダンブラウザ（Chrome / Edge / Firefox / Safari など）で直接開くだけでプレイできます。ビルドやサーバーの起動は不要です。

## 操作方法

| キー | 動作 |
| --- | --- |
| ← / → または A / D | 左右移動（加速・減速あり） |
| Space / W / ↑ | ジャンプ |
| P または Esc | 一時停止 / 再開 |
| Enter または Space | タイトル画面でスタート、ゲームオーバー・クリア画面でタイトルに戻る |

操作方法はタイトル画面内にも表示されます。

## 実装した機能

- 慣性のある水平移動（加速・減速・最高速度）と重力・ジャンプ
- 地面・floating プラットフォーム・壁との当たり判定（軸ごとに解決し、すり抜けを防止）
- プレイヤーに追従するスクロールカメラ（レベル境界でクランプ）
- 高低差のある足場を含む1ステージ分のレベルデザイン
- 落下すると1ライフを失う「穴（ピット）」を複数配置
- クリスタルの収集アイテム（スコア +10）
- 地上を往復する敵、正弦波軌道で飛行する敵の2種類
- 敵の上から踏みつけると撃破（スコア +50）、側面から接触するとダメージ
- スコアシステムとライフ（3ライフ）表示
- タイトル画面（開始前の明確な状態）
- ゲームオーバー画面 / ステージクリア画面
- ページをリロードせずに再スタート可能
- Web Audio API によるジャンプ・収集・撃破・被弾・クリア・ゲームオーバーの効果音（すべて実行時に合成）
- プレイヤーと敵の簡易アニメーション（脚・翼の動き、被弾時の点滅）
- 視差スクロールする背景（星・洞窟のシルエット）
- `requestAnimationFrame` とデルタタイムを用いた更新・描画の分離
- ウィンドウサイズに応じたレターボックス方式のレスポンシブ表示

## ファイル構成

```
index.html   # ページ構造とキャンバス
style.css    # 画面レイアウト・スケーリング用スタイル
game.js      # ゲームロジック・描画・入力・音声すべてを含むスクリプト
README.md    # このファイル
```

## 既知の制限

- レベルは1ステージのみで、セーブ機能はありません。
- 敵はシンプルな往復・正弦波パターンのみで、複雑なAIは実装していません。
- サウンドはすべて `OscillatorNode` によるシンプルな合成音です。
- モバイル端末向けのタッチ操作には対応していません（キーボード操作のみ）。
