# Lumen Run

Lumen Run is an original, browser-based 2D side-scrolling platform game. No external assets, libraries, fonts, or build steps are required.

## Run

Open `index.html` directly in a modern browser. Click the game or press **Enter** on the title screen to begin.

## Controls

- **Left / Right arrows** or **A / D**: move
- **Space**, **W**, or **Up arrow**: jump
- **P** or **Escape**: pause
- **Enter**: start, restart, or return to the title screen

## Features

Acceleration-based movement, jumping and gravity, solid platform collision, a camera that follows the player, collectible light motes, roaming sentinel enemies, enemy-stomping, pits, score, three health points, title/pause/game-over/victory screens, responsive canvas scaling, and Web Audio API sound effects.

## Files

- `index.html` — page shell and canvas
- `style.css` — responsive layout and presentation
- `game.js` — game state, physics, rendering, input, and sound

## Known limitations

The game is intentionally self-contained and uses procedurally drawn canvas art, so it has no save system and no synthesized music track.
