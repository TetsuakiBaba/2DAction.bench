# Lumen Run — 星灯りの回廊

外部素材やライブラリを使わずに制作した、オリジナルの横スクロール2Dプラットフォームゲームです。

## 起動方法

`index.html` をモダンブラウザで直接開いてください。Webサーバー、インストール、ビルドは不要です。

## 操作

- 移動: `←` / `→` または `A` / `D`
- ジャンプ: `Space` / `W` / `↑`
- 一時停止・再開: `P` / `Escape`
- ゲーム開始: `Enter` または画面内ボタン
- ゲーム終了後の再挑戦: `R` / `Enter` または画面内ボタン
- タッチ端末: 画面下部に移動・ジャンプボタンを表示

## 実装内容

- 加速・減速、重力、可変ジャンプ、コヨーテタイム、ジャンプ先行入力
- 床・壁・足場との軸分離衝突判定と追従カメラ
- 高低差のある全1ステージ、複数の落下地点、ゴールゲート
- 20個の光のかけら、巡回する敵、踏みつけ攻撃
- スコア、残機、タイトル、ポーズ、ゲームオーバー、クリア、再スタート
- Canvasによるオリジナル描画、パララックス背景、キャラクターアニメーション、粒子演出
- Web Audio APIによる合成効果音
- キーボード・タッチ操作とレスポンシブ表示
- `requestAnimationFrame` と固定タイムステップによる更新・描画分離

## ファイル構成

- `index.html`: ゲーム画面とインターフェース
- `style.css`: レイアウト、HUD、各状態画面、レスポンシブ表示
- `game.js`: ゲームループ、物理、衝突、敵、描画、音、入力処理
- `README.md`: 起動方法と仕様

## 既知の制限

- 進行状況はブラウザを閉じると保存されません。
- 効果音はブラウザの自動再生制限に従い、最初の操作後に有効になります。
