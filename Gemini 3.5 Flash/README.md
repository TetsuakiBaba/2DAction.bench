# Cyber Jump: Neon Run

ネオン輝く近未来のサイバーワールドを駆け抜ける、完全オリジナルの2D横スクロールプラットフォームゲームです。
HTML5 Canvas, CSS, Vanilla JavaScriptのみを使用して実装されています（外部フレームワーク、ライブラリ、外部アセット等は一切不使用）。

## 実行方法

1. このプロジェクトのディレクトリに移動します。
2. ディレクトリ内にある [index.html](index.html) を任意のモダンブラウザ（Chrome, Firefox, Safari, Edge等）で直接開くだけでプレイが可能です。
   - `file:///` プロトコル、あるいはローカルの簡易HTTPサーバー等を用いて起動できます。

## 操作方法 (Controls)

キーボードによる快適な操作をサポートしています。
画面内のタイトル画面やHUDにも操作ガイドが表示されています。

- **左 / 右移動**: `A` / `D` キー、または `←` / `→` (矢印) キー
- **ジャンプ**: `Space` キー、`W` キー、または `↑` (矢印) キー
- **ゲーム一時停止 / 再開**: `P` キー または `Esc` キー

## 実装されている機能

- **物理エンジン**: 加速度・減衰・重力・プラットフォーム/壁衝突判定、及び崖下落下チェック。
- **カメラ制御**: プレイヤーキャラクターをカメラが自動で横方向に追従（パララックス背景スクロールも実装）。
- **オリジナルデザイン**:
  - **プレイヤー**: ピクセルアート風・ネオン調に輝くドットロボット。歩行アニメーション・空中＆被弾状態での表現を自前描画。
  - **敵キャラクター**: 左右を自動で巡回するネオンドローン。上部から踏みつけることで撃破可能。
  - **ギミック・ハザード**: トゲ（危険地帯）は乗るとダメージが発生し無敵時間に遷移。穴に落ちるとHPが減った状態でスタート地点にリスポーン。
  - **アイテム**: 浮かんで輝くサイバーコイン、回収するとスコアが増加。
- **音響効果 (Web Audio API)**:
  - ジャンプ音、コイン回収、被弾、ゲームクリア時にシンセサイザー音をその場で合成・再生。
- **画面ステート管理**:
  - タイトル画面、プレイ画面、一時停止画面、ゲームオーバー、ステージクリア画面を統合管理。リロード無しでゲームを何度でもやり直し可能。

## 既知の制限事項

- 非常に古いブラウザや、一部のモバイル端末では Web Audio API の自動再生ポリシー制限により、ゲーム開始時に一度画面（あるいはSTARTボタン）をクリックしないと音声が出力されない場合があります。
